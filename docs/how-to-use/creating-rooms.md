# Creating Rooms

## Create Custom Markets

Anyone can create prediction markets on ROOMS in seconds.

## Step-by-Step Guide

### 1. Start Creation
- Click **Create Room** in rooms list
- Or use `/create` command

### 2. Choose Market Type
- **Coin Price** - Track any cryptocurrency price
- **Pump.fun Market Cap** - Track token market cap

### 3. Set Your Question
- Clear, specific title
- Include timeframe if relevant
- Use correct token symbols

### 4. Set Target
- **Price markets:** USD price target
- **Market cap:** Market cap target (e.g., "5m", "10m")
- Be realistic and achievable

### 5. Select Settlement Time
- **Quick:** 15 minutes
- **Short:** 30 minutes
- **Medium:** 1 hour
- **Long:** 2-4 hours
- **Custom:** Your own timeframe

### 6. Deploy
- Review all settings
- Click **Deploy Room**
- Your room goes live instantly!

## Market Type Details

### Coin Price Markets
- Track any cryptocurrency
- Set USD price target
- Choose price or market cap tracking
- Examples:
  - "Will BTC reach $100k?"
  - "Will ETH hit $5k in the next hour?"

### Pump.fun Markets
- Track Pump.fun token market cap
- Enter contract address
- Set market cap target
- Real-time updates every 3 seconds
- Examples:
  - "Will $TOKEN hit $10M market cap?"
  - "Will $PEPE reach $50M?"

## Best Practices

### Good Room Titles
✅ Clear and specific  
✅ Include timeframe  
✅ Use correct symbols  
✅ Realistic targets  

### Bad Room Titles
❌ Vague or unclear  
❌ Missing details  
❌ Wrong token/price  
❌ Impossible targets  

## Host Rewards

## Tips for Success

1. **Popular assets** - More bettors = bigger pools
2. **Realistic targets** - Too easy/hard reduces interest
3. **Clear titles** - Help users understand
4. **Good timing** - Create when interest is high
5. **Engage** - Monitor and interact with bettors
